## Usage
 - Note: Only Windows is officially supported.
 1. Download source code `git clone https://github.com/Anonoei/cpp-opengl-games`
 2. Open `cpp-opengl-games/scripts` folder
 3. Run Win-Setup.bat
 4. Open Solution file
